verifica se dois conjuntos de pontos se sobrepõem.

Utiliza a ordenação nativa da estrutura set para calcular a interseção linear entre duas gamas de tempo de vida de variáveis.

Parameters
set1Primeiro conjunto de pontos
set2Segundo conjunto de pontos
Returns
true Se houver sobreposição em pelo menos uma linha, false caso contrário.
Note
Complexidade Temporal: O(M + N), M e N são os tamanhos dos conjuntos de pontos.
